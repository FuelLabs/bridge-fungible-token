# Fuel Messaging Bridge Integration Tests

Integration tests for the Fuel Messaging Bridge.

## Building From Source

### Dependencies

| dep     | version                                                  |
| ------- | -------------------------------------------------------- |
| Node.js | [>=v14.0.0](https://nodejs.org/en/blog/release/v14.0.0/) |

### Building

Install dependencies:

```sh
npm ci
```

Run tests:

```sh
npm test
```
